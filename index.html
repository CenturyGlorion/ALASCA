ALASCA by OneManMobile

The Android Layout Scaler. Have your apps look right on tablets, small devices and everything in between.

A simple java application with a graphical interface that allows you to choose the layout files in a folder, have their values scaled appropriately and returned as files in a new folder.

Java installed
